## ClearPos <!-- {docsify-ignore} -->

#### Info
> Clears a portion of the console<br>
#### Syntax
`clearPos(int x1, int y1, int x2, int y2);`
* *x1* - the starting x 
* *y1* - the starting y
* *x2* - the ending x 
* *y2* - the edning y
#### Example
```C++
UI::Color("Temporary block\nof randoom\ntexts", 12);
Sleep(1000);
UI::clearPos(1, 0, 5, 2);
```
